# 🧬 PyTorch Kolkata Community – Computational Biology with PyTorch

## Overview

This repository contains educational PyTorch implementations demonstrating how Deep Learning, Graph Neural Networks, Transformers, Autoencoders, and Scientific AI techniques can be applied to Computational Biology, Bioinformatics, Healthcare AI, Neuroscience, and Epidemiology.

All examples are designed for learning purposes and include working code, architecture diagrams, visualizations, and explanations.
